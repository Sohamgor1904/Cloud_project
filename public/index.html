📐 Core Business Logic

Priority Formula
priority = (fill_pct × 0.1) + random(0, 2)

Range: 0–12. 80% fill = 8.0 base + up to 2 urgency = max 10.

Route Generation
Step 1: candidates = bins.filter(priority ≥ 7.0).top10
Step 2: greedyOptimizeRoute(candidates)
score = 0.6 × normFill − 0.4 × normDist

Candidates filtered by priority, then visit order optimised by a greedy nearest-neighbour algorithm: 60% waste urgency + 40% travel proximity using real Haversine distances.

Alert Threshold
fill_pct > 90 → OVERFLOW_RISK
fill_pct > 75 → WARNING
fill_pct > 40 → MONITOR

Member 4 alert engine triggers on >90%. Frontend renders banner.

Fill Colour Rules
< 40% → 🟢 Green (skip)
40–75% → 🟡 Amber (monitor)
> 75% → 🔴 Red (in route)

Applied to map markers, table badges, route cards, and charts.
